# 하늘결 프로젝트
> **구름을 읽는 시간**

```
  "구름에 관심이 있는 사람들도 구름을 설명해주는 사이트가 있으면 얼마나 좋을까?"
  라는 사고에서 출발한 이 프로젝트는 구름의 종, 변종, 부속 구름, 생성 원리, 분류 기준 등을 체계적으로 정리하여 제공합니다.
  국제구름사진의 한글 자료 부족을 느끼고, 누구나 쉽게 구름을 배우고 이해할 수 있는 플랫폼을 만들고자 시작되었습니다.
```
